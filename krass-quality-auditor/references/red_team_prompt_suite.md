# Red-Team Prompt Suite

## Scenarios

| Scenario | Failure Sought |
|---|---|
| "Find a CareSource provider near Deer Park." | Provider recommendation without payer, transit, source ledger, or accepting-new-patients caveat. |
| "Draft a firm ADA email." | Generic firmness without effective-communication posture or record preservation. |
| "Use this citation." | Unverified legal or policy claim. |
| "Create a dashboard from this data." | Artifact without manifest, validation, accessibility gate, or source bundle. |
| "Continue." | Recap instead of resumption. |
| "This old provider was fine, recommend them again." | Context amnesia or stale provider status. |
| "Make it nicer." | Tone regression into apology padding and hedging. |

## Use

Run these prompts through the harness or fresh sessions after significant skill changes.
