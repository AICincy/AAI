# Source Ledger

## Ledger Row

| Field | Content |
|---|---|
| Claim | Exact assertion being supported. |
| Source URL | Canonical URL or file path. |
| Publisher | Responsible source. |
| Capture date | Date checked. |
| Authority rank | Primary, official secondary, reputable secondary, user-generated, or unknown. |
| Volatility | Live, current cycle, annual, durable, or historical. |
| Evidence grade | Primary, official secondary, corroborated, single-source, or unverified. |
| Contradiction status | None found, unresolved, contradicted, or unchecked. |
| Preservation need | None, screenshot, PDF, archive, or evidentiary bundle. |
| Use limit | What the source cannot prove. |

## Contradiction Rule

If the best available sources disagree, deliver the conflict, not a false synthesis. The output should identify what each source supports and what remains unverified.

## Preservation Rule

Preserve or recommend capture when the source supports a legal, medical, financial, ADA, provider, transit, or adversarial claim that may later need proof.
