# Live Source Profiles

## Current Synthesis

Some domains age faster than others. Apply the strictest recency requirement that fits the claim.

## Profiles

| Domain | Primary Source | Recency Requirement | Notes |
|---|---|---|---|
| OpenAI Codex and skills | Official OpenAI documentation plus local installed files | Current cycle | Product behavior can change quickly. |
| ADA communication | ADA.gov plus controlling law or policy when needed | Durable unless law or policy changed | Guidance may be nonbinding; state limits. |
| WCAG cognitive accessibility | W3C WCAG 2.2 Understanding and normative WCAG text | Durable until version change | Use for accessibility gates, not medical framing. |
| CareSource provider network | CareSource Find A Doctor/Provider, member documents, provider portal, or direct provider confirmation | Live | Directory listings can be stale. |
| Cincinnati fixed-route Metro | Metro official route, schedule, fare, and alert pages | Live or service date | Check service day and direction. |
| MetroNow | MetroNow official page, app, or phone booking information | Live | Do not treat it as fixed-route bus service. |
| Metro Access | Metro official Access information | Current cycle | Eligibility and booking constraints control feasibility. |
| GTFS schedule | Metro developer data and GTFS feed | Current feed cycle | Store feed or page update date. |
| GTFS Realtime | GTFS Realtime reference and available agency feeds | Live | Separate alerts, trip updates, and vehicle positions. |
| Prices and fees | Official vendor, agency, payer, or institution | Live | Do not infer affordability without checking. |

## Recency Labels

| Label | Meaning |
|---|---|
| Live | Verify in the current task before output. |
| Current cycle | Verify during the current policy, product, or schedule cycle. |
| Annual | Verify against the current calendar, school, benefit, or plan year. |
| Durable | Stable until law, standard, or product version changes. |
| Historical | Use only for past-state analysis. |
