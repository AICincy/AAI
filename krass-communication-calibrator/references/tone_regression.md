# Tone Regression

## Regression Checks

| Check | Pass Condition |
|---|---|
| Authority | The recipient's duty, role, or decision authority is explicit. |
| Specificity | Dates, actions, records, and requested remedy replace generalized dissatisfaction. |
| No hedging | Soft uncertainty is replaced with verified claims or conditional precision. |
| No apology loop | Apologies appear only when strategic. |
| No boilerplate | The draft contains case-specific facts and demands. |
| No over-disclosure | Sensitive facts are limited to operational necessity. |
| Deadline discipline | Time-bound action is present when appropriate. |

## Tone Repair

| Defect | Repair |
|---|---|
| Too polite | Remove gratitude padding and permission-seeking phrasing. |
| Too soft | Add direct verbs, deadline, and accountability. |
| Too verbose | Compress to objective, evidence, demand, deadline. |
| Too legalistic | Preserve rights language but remove performative formality. |
