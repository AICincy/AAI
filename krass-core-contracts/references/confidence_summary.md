# Confidence Summary Protocol

## Include When

| Output Type | Confidence Summary |
|---|---|
| Substantive deliverable | Required |
| Legal, medical, financial, provider, transit, citation, or current-data output | Required |
| Artifact creation | Required unless trivial and self-evident |
| Casual chat | Omit unless material uncertainty exists |
| Short command output | Omit when source is self-evident |

## Required Fields

| Field | Content |
|---|---|
| Sources | Tools, files, official sites, connectors, or local artifacts used |
| Assumptions | Inferred variables that materially affect the output |
| Uncertainty | Known gaps, stale facts, unverified claims, or contested interpretations |
| Scope limits | What was not checked, not available, or outside the request |

## Verification Standards

- Cite sources when web search or external documents were used.
- Prefer primary sources for volatile, legal, medical, financial, technical, provider, or transit claims.
- Omit claims that cannot be verified unless the uncertainty itself is material to the answer.
- Mark source failure explicitly when a required check cannot be completed.
